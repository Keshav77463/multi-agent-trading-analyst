# Multi-Agent Trading Analyst

This project implements a **multi-agent stock analysis system** using CrewAI.  
Multiple AI agents collaborate to analyze stock market data and provide trading recommendations.

## 🚀 Features

- Stock data analysis using real-time market information
- Multi-agent collaboration
- Buy / Sell / Hold recommendations
- Modular AI agent architecture

## 🧠 Agents

### 1. Analyst Agent
- Retrieves stock market data
- Analyzes price movement and market indicators

### 2. Trader Agent
- Uses the analysis results
- Makes a **BUY / SELL / HOLD** decision

## ⚙️ Technologies Used

- Python
- CrewAI
- Groq LLM
- yfinance
- Python-dotenv
